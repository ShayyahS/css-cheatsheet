Common CSS properties and what they do

This website will go over the most common CSS properties used in web development and explain what they are used for.

Tag Name Description Example
background-color: ; Background Colour This property controls the colour of the background of the specified element. This can be useful for improving readability or improving the aesthetic of the element. background-color: red;
background-image: ; Background Image This property allows the developer to use an image as the background for a specific element.
border: ; Border This property controls the styling of the border around the specified element. Green ridged border
color: ; Colour This property controls the colour of the text in the specified element. Yellow coloured font
font-family: ; Font Family This property controls the typeface of the text specified by the element. The font-family of the website is IBM Mono Plex. This cell is Verdana.
font-style: ; Font Style This property controls the style of the font, such as italics, boldness or normal. The text in this cell is italicised.
margin: ; Margin This controls the amount of empty space on the outside of the element (outside of the padding and border). Margin is also useful for centering elements by making use of auto. Tables can't have margin, so here is a picture to illustrate what margin does. (the margin of this image is set to margin: 0 auto; which centres it within the container.
opacity: ; Opacity This controls the transparency of the element. This cell has 50% opacity.
padding: ; Padding This controls the amount of space between the element itself and the border (or margin fi there is no border). This text has 50 padding.
text-align: ; Text Alignment This controls the alignment of the text. This text is aligned to the right of the cell.
text-decoration: ; Text Decoration This controls the decoration applied to the text. This is especially useful for removing the blue and underline decoration which is applied to hyperlinks by default. This text has default text decoration.

 This text has no text decoration. They are both hyperlinks.
transform: ; Transform This lets the developer apply transformations to the element, such as rotation, translation, scaling, skewing or matrix.

This text has been transformed.
